# No-Scroll UI Update

This build focuses on faster daily use and a cleaner single-screen workflow.

## Changes

- Rebuilt the main workspace to avoid vertical scrolling during normal use.
- Reduced oversized hero/header height.
- Converted the Targets screen into a two-column layout:
  - left: Add Target
  - right: Target Queue
  - bottom: One-Click Tools
- Made the status console compact so it does not consume the screen.
- Kept the interface English-only.
- Improved spacing, button size, card density, and table visibility.
- Evidence Locker now keeps note controls and evidence table on one screen.
- Dropdowns remain compact and no longer overlap form fields.

## Recommended resolution

The UI is optimized for 1280x800 and above. For smaller screens, maximize the window.
