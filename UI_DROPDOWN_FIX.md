# Final UI Dropdown Fix

This build fixes the macOS/retina dropdown rendering issue where combo-box menus could appear oversized or overlap the form.

## Fixed

- Scoped card styling so it no longer leaks into combo-box popup windows.
- Rebuilt target and evidence forms with a stable grid layout instead of wide form labels.
- Added compact combo-box popup sizing for Type, Priority, and Status fields.
- Improved dark contrast and spacing around inputs.
- Kept the application UI English-only.

## Recommended usage

Open the app, set one Active Case, then use:

1. Overview
2. Targets & Collect
3. Analyze & Report
4. Evidence
