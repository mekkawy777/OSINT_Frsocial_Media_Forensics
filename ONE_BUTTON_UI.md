# One Button Workspace

This edition removes the crowded tabs and tool buttons. The GUI now has a single main workflow:

1. Enter one **Active Case**.
2. Paste one or more public targets, one per line.
3. Click **START FULL INVESTIGATION**.

The single action automatically:

- Normalizes the case ID.
- Adds all targets to the case.
- Blocks duplicate targets.
- Detects target type automatically: URL, domain, username, or profile.
- Collects public evidence.
- Extracts public profile/page signals.
- Runs username search for username targets.
- Builds the network graph.
- Generates the HTML/Markdown reports.
- Verifies evidence hashes.
- Exports CSV, manifest, and ZIP bundle.
- Opens the final HTML report.

The app remains English-only and public-OSINT only. It does not bypass login, CAPTCHA, paywalls, or private/restricted content.
